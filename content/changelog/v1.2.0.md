## Changed
- Refined icon category taxonomy to improve clarity, scalability, and discoverability.
- Narrowed the scope of the general category to focus on core UI utilities and system level primitives.
- Re scoped symbol to represent semantic meaning, status, warnings, reactions, and abstract concepts rather than functional controls.
- Reclassified multiple icons across categories to better align with their primary intent and usage context.

## Notes
- No icons were removed in this release.
- This release focuses on structural consistency rather than visual changes.
- Existing icon names and styles remain unchanged.
