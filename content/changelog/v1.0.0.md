## Added
- First stable public release of the Some Icons icon set
- Initial curated set of icons released under a unified design system
- Outline and filled styles available at launch
- Core categories and tags implemented for search and filtering

## Design System
- Consistent grid and geometry across all icons
- Harmonized stroke weight and corner radii between styles
- Designed for clarity and legibility at common UI sizes

## Supported Platforms
- Website library browser
- Figma plugin for direct insertion and export

- Some icons are still undergoing refinement and may receive visual adjustments in future releases
- Icon names and IDs introduced in this release are considered stable unless explicitly noted in future changelogs
